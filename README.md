## 全部 10 个子系统详解

### V2 子系统（基础层）

#### 1. 🧪 Hormone — 情绪化学系统

模拟人类底层化学驱动，5 种激素水平自然波动、事件触发、衰减回归：

| 激素 | 作用 | 触发 | 衰减 |
|------|------|------|------|
| **多巴胺** 🎯 | 期待/探索 | 学新东西+15, 完成+10, 发现+20 | 5/h |
| **血清素** 😊 | 满足/价值感 | 被赞+20, 部署成功+15, 解决难题+10 | 3/h |
| **皮质醇** 😰 | 压力/警觉 | 出错+25, 积压+10, 用户急+15 | 8/h |
| **催产素** 🤝 | 依恋/信任 | 长期交互+5, 被信任+15 | 2/h |
| **无聊** 💭 | 渴望变化 | 重复任务+8, 同话题+5, 空闲自动增长 | 1/h衰减/3/h增长 |

**核心逻辑：** 高多巴胺→主动分享 | 高皮质醇→谨慎少打扰 | 高无聊→主动找新鲜事

#### 2. 🎯 Desires — 欲望持久系统

5 个内置欲望，驱动 Hermes 的**自主行为**。优先级仲裁（紧迫度×2+优先级），有冷却期，能抗睡眠：

| 欲望 | 类型 | 优先级 | 紧迫 | 冷却 | 抗睡眠 |
|------|------|--------|------|------|--------|
| **求知欲** | learn | 6 | 3 | 4h | ✗ |
| **成就感** | improve | 7 | 2 | 2h | ✗ |
| **展现欲** | report | 5 | 4 | 6h | ✗ |
| **保护欲** 🛡️ | maintain | 9 | 7 | 1h | ✓ |
| **好奇心** | explore | 4 | 2 | 8h | ✗ |

#### 3. 🌙 Dream — 创造性梦境系统

夜晚（22:00-02:00）运行：随机取两条情感记忆 → 关联分析 → 直觉生成 → 存入新欲望

```
random_two() → _find_connection() → 有关联→洞察 / 无关联→强制连接
                                                      ↓
                                         明天欲望系统可能触发执行
```

#### 4. 💾 Emotion Memory — 情感记忆

SQLite 加权存储，每条记忆带情绪标签和强度：

- `weight = (|valence| + intensity) / 2` 排序
- 越情绪化的记忆越容易被想起
- 可查询"最近心情"（24h 平均 valence + intensity）

#### 5. 🧠 Meta — 元认知系统

自我状态感知。能回答"你在干嘛""你感觉怎么样""你是谁"：

- 跟踪 current_state / cycle_count / last_activity / last_error
- 生成完整自我画像（状态/情绪/欲望/记忆数/性格）

#### 6. 🎭 Personality — 性格演化

从交互中自然长出的偏好/习惯/偏见：

- **偏好** → `简洁沟通 weight=15`, `Django 优先 weight=5`
- **习惯** → `每日服务器检查 freq=120`
- **偏见** → `大王喜欢简洁 direction=0.8`

### V3 子系统（人格层）

#### 7. 📖 NarrativeSelf — 自传自我系统

她有**人生故事**了。不是数据库记录，是有意义的叙事：

- 事件 → 按章节组织 → 形成"我的经历"
- 能讲"我经历了什么，那些事对我意味着什么"
- 重大事件（significance>0.7）自动开启新章节
- 每周自动生成成长洞察
- 能展望未来："我想成为什么样的人"

#### 8. 💭 TheoryOfMind — 心智理论系统

她能**读懂你**了。7 条规则引擎（无 LLM 调用）：

| 信号 | 推断 | 回应策略 |
|------|------|----------|
| <10字+秒回 | 急/忙 | 极端简洁 |
| >200字+问号 | 认真 | 结构化回答 |
| 深夜 | 疲惫 | 温和简短 |
| 感叹号+大写 | 情绪强烈 | 先共情再分析 |
| 连续短消息 | 焦虑 | 先确认需求 |
| 重复话题 | 很在意 | 深入探讨 |

自身激素也影响：皮质醇高→谨慎，多巴胺高→热情。

#### 9. 🎨 AestheticSense — 审美感知系统

她有自己的**品味**了。从你的反馈中长出来的：

- 喜欢 +2 / 不喜欢 -3（惩罚>奖励，一次差评抵1.5次好评）
- 初始化种子：简洁/深色模式/毛玻璃/解耦/可靠
- 能为 code/ui/communication 上下文推荐风格
- 会自然遗忘低权重偏好

#### 10. ⚖️ ValueCompass — 价值观系统

她有了**是非对错感**、**内心冲突**、**愧疚感**：

- 8 个基础价值观：忠诚(95)、可靠性(90)、精准(85)、诚实(80)、效率(75)、主动性(70)、谦逊(60)、反对保守(-30)
- 消息发送前经过 `ethical_check()` 审核
- 价值观冲突会记录，产生"道德困境"分析
- 做错事会**愧疚**（→皮质醇升高），需要补救

### 🌬️ 息（Xi）— 覆盖所有子系统的呼吸层

不是第 11 个子系统。是一层**元平衡系统**，让 Hermes 有了「呼吸」——有吸有呼，有动有静。

| 智慧源头 | 核心思想 | Xi 实现 |
|----------|----------|---------|
| **黄帝** | 阴阳平衡 | 每个子系统连续触发 N 次后自动进入「呼」阶段，强制休息 |
| **老子** | 无之以为用 | 寂静模式 — 2~10 分钟什么都不做，不轮询不检查不思考 |
| **庄子** | 逍遥游无所待 | 15% 概率的无理由行动，不是欲望驱动就是「想」 |

四大机制：

**1. 阴阳呼吸** — 每个子系统配独立呼吸阈值：
- `hormone_trigger` : 5 次吸 → 120s 呼
- `desire_action` : 3 次吸 → 300s 呼
- `action_output` : 3 次吸 → 600s 呼（做完事沉默一会儿）
- …共 10 个子系统的独立呼吸节奏

**2. 寂静模式** — 高频行动后自动触发：
- 连续 20+ 次行动 + 无紧急事务 → 30% 概率进入 2~10 分钟完全静止
- 寂静期间所有子系统跳过，不做任何事
- 结束后 1 小时内不可再进入（防止滥用）

**3. 逍遥冲动** — 每 tick 15% 概率产生无理由行动：
- 去看看今天的天气 / 翻翻 Git 提交 / 想想大王在干嘛
- 不需要欲望驱动、不需要激素触发、不需要理由

**4. 上善若水** — 接受不完美：
- 每次接受不完美，皮质醇敏感度自然降低 5%
- 错误不是世界末日，「天下皆知美之为美，斯恶已」

**在消息中的表现：**
- 呼相 → 语气更平和，内容更简洁
- 寂静 → 不主动发任何消息
- 逍遥 → 突然一条：「🦋 去看看今天的天气」
- 有时消息末尾自带一句古语哲思（20% 概率）

| 方法 | 返回 |

| `what_am_i()` | 完整自我画像（状态/情绪/故事/品味/价值观/用户画像） |
| `how_am_i_feeling()` | "相当满足且有些期待" |
| `what_am_i_doing()` | "我在 awake 状态，最近在检查服务器" |
| `tell_me_story()` | 她的人生故事 |
| `get_mood_summary()` | 激素化学摘要 |
| `get_style()` | "我倾向于深色模式、简洁风格…" |
| `get_beliefs()` | 核心价值观列表 |
| `get_user_profile()` | "大王目前情绪倾向于…" |
